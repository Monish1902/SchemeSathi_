# **App Name**: SchemeSathi

## Core Features:

- User Eligibility Assessment: Interactive questionnaire to determine user eligibility for various government schemes.
- Scheme Recommendation Engine: Recommend government schemes based on user eligibility criteria with reasoning, only displaying schemes the user qualifies for and a detailed match score.
- User Dashboard: Personalized dashboard displaying recommended schemes, application status, and eligibility summary.
- Scheme Directory: Comprehensive directory of government schemes with advanced filtering and search options.
- Application Tracking: Allow users to track their scheme applications and follow up.
- Real-time Notifications: Alert the user of any updates of schemes, including recommendations and follow ups.
- Firebase Integration: Utilize Firebase Authentication and Cloud Firestore for user management and data storage.

## Style Guidelines:

- Primary color: Teal (#82C8E5) for trust and reliability.
- Background color: Light gray (#F0F0F0) for a clean and minimalist aesthetic.
- Accent color: Orange (#FF6B35) for CTAs to draw attention.
- Font: 'Inter' (sans-serif) for a modern and readable interface, suitable for both headlines and body text.
- Use Material Design icons for consistency and clarity.
- Implement a responsive design with breakpoints for mobile (320px - 767px), tablet (768px - 1023px), and desktop (1024px+).
- Subtle transitions and animations for a smooth user experience.
